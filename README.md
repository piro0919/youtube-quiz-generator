# YouTube Quiz Battle

> Generate quizzes from YouTube videos and battle with friends.

[🔗 Live Site](https://youtube-quiz-battle.kkweb.io/)

## ✨ Features

- 🎥 Paste a YouTube URL to auto-generate a quiz
- 🤖 AI-powered question generation
- 👥 Battle mode for head-to-head matches
- 📊 Score tracking

## 🛠 Tech Stack

- Next.js + React + TypeScript
- AI SDK (OpenAI / Anthropic)
- Tailwind CSS

## 🚀 Development

```bash
pnpm install
pnpm dev
```
